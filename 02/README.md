## 编程简史-PHP是世界上“最”好的语言_哔哩哔哩_bilibili


* 2003年, WordPress发布
* PHP-FPM
* Laravel
* Facebook 用 C++开发了HHVM
* Swoole 使用C语言开发
* PHP 包管理工具 Composer 发布
* 2013 年 Workerman 发布 [开源高性能PHP应用容器](https://www.workerman.net/)
* Laravel 精简之后发布 Lumen, 性能没有提升多少
* Thinkphp5 发布
* [Hyperf | 一款高性能的企业级协程框架，特别适用于微服务架构、中间件、Web系统、API 系统的构建，由 Swoole 和 Swow 引擎提供底层支持](https://hyperf.io/)
* 2018 [RoadRunner](https://roadrunner.dev/) 发布, 是使用GoLang 开发的PHP高性能应用服务器
* Livewire 发布， 是Laravel 社区开发的php库， 他可以说是服务端的Vue
* [FrankenPHP: the modern PHP app server](https://frankenphp.dev/) 可能是在视频号上看的
